How to make a WebSite

What is a WebSite?

A WebSite is a set of related WebPages located under a single domain name, typically produced by a single person or an organization.

A WebSite can also be said as any webpage whose content is identified by a common domain name, and is published on at least one web server.

What is a WebPage?

A webpage is a web document that is accessed in a web browser.

It is or can also be said as a page on a website.

Types of web developers

There are basically 3(three) of web developers, namely:

  1. Frontend Developer
  2. Backend Developer
  3. Full-Stack Developer

Note: In this course, we are going to be teaching only frontend development.
