Introduction To GitHub


Millions Of Developers call GitHub home. Whether you're scaling your development or just learning how to code, GitHub is where you belong.

What is Github?


GitHub is a cloud-based platform where you can store, share, and work together with othere to write code. GitHub is a code hosting platform for collaboration and version control.


GitHub can also be said as a web-based platform that stores Git repositories in the cloud, allowing developers to work together on projects. GitHub is also a cloud-based Git Server for code hosting and social coding, and it implements features for code review, diffs, push & pull requests, etc.

Features Of GitHub


  1. Repositories
  2. Branches
  3. Pull Requests(PRs)
  4. Issues And Bug Tracking
  5. GitHub Actions
  6. Wiki And Documentation
  7. Code Hosting
  8. Easy Collaboration
